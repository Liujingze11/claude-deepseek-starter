# i18n README — English Primary, Chinese Secondary

## Overview

增加英文 README，以英文为主。现有中文 README 改名为 `.zh.md`，新建英文 `README.md`。中英文版本内容一致，顶部互加语言切换链接。

## 文件方案

| 文件 | 内容 | 操作 |
|------|------|------|
| `README.md` | 英文，顶部 `[中文](README.zh.md)` | 新建（翻译当前中文主 README） |
| `README.zh.md` | 中文，顶部 `[English](README.md)` | 重命名当前主 README + 加链接 |
| `macos/README.md` | 英文，顶部 `[中文](README.zh.md)` | 新建（翻译当前精简版） |
| `macos/README.zh.md` | 中文，顶部 `[English](README.md)` | 重命名 + 加链接 |
| `linux/README.md` | 英文，顶部 `[中文](README.zh.md)` | 新建 |
| `linux/README.zh.md` | 中文，顶部 `[English](README.md)` | 重命名 + 加链接 |
| `windows/README.md` | 英文，顶部 `[中文](README.zh.md)` | 新建 |
| `windows/README.zh.md` | 中文，顶部 `[English](README.md)` | 重命名 + 加链接 |

## 语言链接格式

- 英文版顶部：`[中文](README.zh.md)`（子 README 同路径）
- 中文版顶部：`[English](README.md)`（子 README 同路径）

链接放在标题上方第一行。

## 内容要求

- 英文版和中文版内容一一对应，不增不减
- 中文版使用已精简过的当前版本
- `docs/troubleshooting.md` 和 `docs/publish-checklist.md` 不做双语化（仅中文）
