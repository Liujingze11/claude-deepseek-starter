图标方向 R2 — 从「感觉」出发

不考虑 emoji，从真正的图标设计语言来思考

A

对话 + 光标

圆角底 + 一个聊天气泡，气泡末尾是一个闪烁的文本光标 | 。传达"和 AI 对话写代码"。友好、有温度，不硬核。

B

代码括号 + 火花

圆角底 + </> 符号，上方有一颗小火花/星星。传达"AI 帮你写代码"。简洁、现代，像 VS Code 那种工具感。

C

魔棒 + 终端窗口

终端窗口轮廓里伸出一根魔棒，魔棒尖端有闪光。传达"用 AI 魔法操控命令行"。叙事感强，一眼就懂。

D

双色抽象几何

不具象，纯几何：两个交错重叠的圆角方形，深蓝+青绿渐变，中间负空间形成 C 形。像 Slack、Figma 那种现代 SaaS 图标风格。
