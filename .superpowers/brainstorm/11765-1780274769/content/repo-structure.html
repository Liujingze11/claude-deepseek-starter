各格式规格

平台格式包含尺寸文件大小
Windows.ico16, 32, 48, 256 px~30KB
macOS.icns16, 32, 64, 128, 256, 512, 1024 px~50KB
Linux.png128px~5KB

所有二进制文件合计 ~120KB，对仓库体积基本没影响

Windows 额外说明

Windows 目前用 WScript.Shell 创建的 .lnk 已经保存在桌面，只需加一行 IconLocation。setup.bat（双击入口）本身无法自定义图标，但双击后创建的桌面快捷方式可以。

也考虑给 setup.bat 做个包装 .lnk 放到下载包里，用户一开始就看到图标。

macOS 额外说明

目前桌面放的是 .command 纯文本，macOS 不给他图标。改为 .app bundle 后获得完整图标支持。.app 包内只放一个脚本 + 一个 icns，不需要 Xcode。
破坏性评估：.command 换成 .app 后，双击行为不变（打开Terminal选文件夹），需要删掉旧的 .command 避免桌面上出现两个。
